[Action Required: Windows XP OS users] End of Support for 3DES Cipher


For security reasons, we have discontinued the support for 3DES Cipher. After this discontinuation of support for 3DES cipher, machines running on Windows XP, that do not support any more recent and secure ciphers than 3DES, will not be able to access the Sign In and My Account pages.

If you're a Windows XP user, we strongly recommend upgrading your OS.

Video

The Video API manipulates a video element's size, position and orientation.

webOS Signage supports playing only one video object AND one audio element simultaneously at any given time. Thus DO NOT attempt to use more than one video object or one audio element simultaneously.

getVideoStatus

DESCRIPTION

Returns the size and position information of a video element in an HTML page which has called this method.

SINCE
SCAP v1.0
Platform Compatibility
All

SYNTAX

Video.getVideoStatus(successCallback, errorCallback)

PARAMETERS

None

RETURNS

Property Type Description Returned
source Object Contains the size and position information of the video element in the HTML page which has called this method. Always
source.x Number The x-coordinate of the top left corner the video element. Always
source.y Number The y-coordinate of the top left corner of the video element. Always
source.width Number The width of the video element. Always
source.height Number The height of the video element. Always

function getVideoStatus() {
    function successCb(cbObject) {
        console.log("cbObject : " + JSON.stringify(cbObject.source));
 
        console.log("source.x : " + cbObject.source.x);
        console.log("source.y : " + cbObject.source.y);
        console.log("source.width : " + cbObject.source.width);
        console.log("source.height : " + cbObject.source.height);
        // Do something
        ...
    }
 
    function failureCb(cbObject) {
        var errorCode = cbObject.errorCode;
        var errorText = cbObject.errorText;
 
        console.log("Error Code [" + errorCode + "]: " + errorText);
    }
 
    var video = new Video();
    video.getVideoStatus(successCb, failureCb);
}

setVideoSize

DESCRIPTION

Zooms in or out of the video element in the HTML page which has called this method. The given source object defines the area for zooming and the zooming takes place inside the area allocated to the video element. The maximum zoom ratio is 1x zoom.

SINCE
SCAP v1.0
Platform Compatibility
All

SYNTAX

Video.setVideoSize(successCallback, errorCallback, options)

PARAMETERS

Property Type Description Required
source Object The target area for zooming. Contains properties for the size and position information of the area to zoom in or zoom out. Required
source.x Number

The x-coordinate of the top left corner of the area to zoom in or zoom out.

Range: [0–(Video element width - source.width)]

 Required
source.y Number

The y-coordinate of the top left corner of the area to zoom in or zoom out.

Range: [0–(Video element height - source.height)]

 Required
source.width Number

The width of the area to zoom in or zoom out.

Range: [0–(Video element width - source.x)]

 Required
source.height Number

The height of the area to zoom in or zoom out.

Range: [0–(Video element height - source.height)]

 Required

RETURNS

None

REMARK

  • Restriction: Use ONLY ONE video tag per HTML page. Regardless how many video elements are in an HTML page, only one video element gets played.
  • Constraint: The source object properties are constrained by the video element size.
    • The source object's horizontal boundary, i.e. the sum of source.x + source.width, shall not exceed the width of the original video element.
    • The source object's vertical boundary, i.e. the sum of source.y + source.height, shall not exceed the height of the original video element.

function setVideoSize() {
    var options = {
        source: {
            x: 10,
            y: 10,
            width: 900,
            height: 600
        }
    };
 
    function successCb() {
        // Do something
    }
 
    function failureCb(cbObject) {
        var errorCode = cbObject.errorCode;
        var errorText = cbObject.errorText;
 
        console.log("Error Code [" + errorCode + "]: " + errorText);
    }
 
    var video = new Video();
    video.setVideoSize(successCb, failureCb, options);
}
