[Action Required: Windows XP OS users] End of Support for 3DES Cipher


For security reasons, we have discontinued the support for 3DES Cipher. After this discontinuation of support for 3DES cipher, machines running on Windows XP, that do not support any more recent and secure ciphers than 3DES, will not be able to access the Sign In and My Account pages.

If you're a Windows XP user, we strongly recommend upgrading your OS.

PMMode

DESCRIPTION

Contains constant values that represent power management modes. The power management modes determine how a signage device acts when its power is turned off.

SINCE
SCAP v1.4
Platform Compatibility
webOS Signage 3.0 and later

CONSTANT PROPERTIES

See the remark section below for the conditions that trigger a signage device to enter and exit each PM mode.

Name Value Description
Power.PMMode.PowerOff "powerOff" Turns off everything when powering off a signage device. This is the default PM mode.
Power.PMMode.ScreenOff "screenOff" Turns off the screen only with the specific conditions.
Power.PMMode.ScreenOffAlways "screenOffAlways" Turns off the screen only. The trigger for entering and exiting this mode differs from the ScreenOff mode.
Power.PMMode.ScreenOffBacklight "screenOffBacklight" Turns off the screen only. The trigger for entering and exiting this mode differs from the ScreenOffAlways mode.
Power.PMMode.SustainAspectRatio "sustainAspectRatio" Almost identical to the PowerOff mode, except that the switching IC is not turned off.

REMARK

The following table shows the conditions that trigger a signage device to enter to and wake up from the PM Mode.

PM Mode Triggers to Enter the PM Mode Triggers to Wake Up from the PM Mode
PowerOff

  • Pressing the Monitor Off button on a signage device.
  • Power-off timer.
  • The RJ45 command ka, the power off command.
  • The RS232C command ka, the power off command.
  • Auto power off by the Automatic Standby Mode.
  • No-signal power off by the 15-minute-off feature.
  • Abnormal Shutdown. (e.g. due to extream temperature conditions)
  • DPM (Display Power Management) activation.
  • Pressing the power button on a remote control or on a signage device.
  • Pressing the Input button on a remote control or on a signage device.
  • Pressing the Monitor On button on a signage device.
  • Pressing a number key (0-9) on a remote control.
  • Power-on Timer.
  • The RJ45 command ka, the power on command.

  • Receiving a WoL (Wake on Lan) packet.
  • Receiving an external input signal (e.g. HDMI) when the device had been powered off by the DPM.

  • RJ45 commands: (with commands, video and audio features are activated)

SustainAspectRatio
