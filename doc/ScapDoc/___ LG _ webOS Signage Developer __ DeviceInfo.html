[Action Required: Windows XP OS users] End of Support for 3DES Cipher


For security reasons, we have discontinued the support for 3DES Cipher. After this discontinuation of support for 3DES cipher, machines running on Windows XP, that do not support any more recent and secure ciphers than 3DES, will not be able to access the Sign In and My Account pages.

If you're a Windows XP user, we strongly recommend upgrading your OS.

getNetworkInfo

DESCRIPTION

Returns the network information of a signage device. The network information includes IP address assigned to wireless/wired LAN, gateway address, netmask address and DNS address.

SINCE
SCAP v1.0
Platform Compatibility
All

SYNTAX

DeviceInfo.getNetworkInfo(successCallback, errorCallback)

PARAMETERS

None

RETURNS

Property Type Description Returned
isInternetConnectionAvailable Boolean Indicates Internet connection availability.
  • true: Internet connection available
  • false: Internet connection unavailable
 Always
wired Object Contains information on a wired network. Always
wired.state String Indicates whether a signage device is connected to a wired network.
Range: ["connected", "disconnected"]		 Always
wired.interfaceName String The name of the wired network interface. Optionally
wired.ipAddress String The IP address assigned to the wired network interface. Optionally
wired.netmask String The netmask address assigned to the wired network interface. Optionally
wired.gateway String The gateway address assigned to the wired network interface. Optionally
wired.onInternet String

Indicates whether a signage device is connected to the Internet on a wired network.

Range: [ "yes", "no"]

 Optionally
wired.method String Indicates the IP allocation method for the wired network.
  • "dhcp": Dynamic allocation.
  • "manual": Fixed allocation.
 Optionally
wired.dns1 String The DNS address assigned to the wired network interface Optionally
wired.dns2 String The secondary DNS address assigned to the wired network interface. Optionally
wired.dns3 String Tertiary DNS address assigned to the wired network interface Optionally
wired.dns4 String Quadruplicative DNS address assigned to the wired network interface Optionally
wired.dns5 String Fifty-fifth DNS address assigned to the wired network interface Optionally
ipv6 Object IPv6 configuration. (only present when context is configured with IPv6) Optionally
ipv6.gateway String Gateway IPv6 address assigned to the wired network interface. Optionally
ipv6.ipAddress String IPv6 address assigned to the wired network interface. Optionally
ipv6.prefixLength String The prefixLength for the IPv6 address. Optionally
ipv6.method String IP allocation "manual": fixed allocation "dhcp": dynamic allocation. Optionally
wifi Object Contains information on a wireless network. Always
wifi.state String

Indicates whether a signage device is connected to a wireless network.

Range: ["connected", "disconnected"]

 Always
wifi.interfaceName String The name of the wireless network interface. Optionally
wifi.ipAddress String The IP address assigned to the wireless network interface. Optionally
wifi.netmask String The netmask address assigned to the wireless network interface. Optionally
wifi.gateway String The gateway address assigned to the wireless network interface. Optionally
wifi.onInternet String

Indicates whether the device is connected to the Internet on a wireless network.

Range: [ "yes", "no"]

 Optionally
wifi.method String Indicates the IP allocation method for the wireless network.
  • "dhcp": Dynamic allocation.
  • "manual": Fixed allocation.
 Optionally
wifi.dns1 String The DNS address assigned to the wireless network interface. Optionally
wifi.dns2 String The secondary DNS address assigned to the wireless network interface. Optionally

REMARK

  • The properties describing a network (e.g. interfaceName) will be returned only if a signage device is connected to network. For example, the wired.method property will be returned if a signage device is connected—i.e. wired.state has the value "connected"—to a wired network.
  • If a signage device has no built-in Wi-Fi, Wi-Fi related return values will be returned only if the signage device has an external Wi-Fi device connected.
